# mylife 交互状态说明

更新时间：`2026-06-30`

## 主窗口默认态

- 打开默认进入 `今天`
- 中间区域按状态分组展示今天任务
- 右侧默认显示最近一次选中任务详情
- 若无选中任务，右侧显示空态与快捷入口

## 任务生命周期

状态：

- `未开始`
- `进行中`
- `搁置`
- `已完成`
- `废弃`

规则：

- 简单任务主路径：`创建 -> 完成`
- 复杂任务允许父子孙层级
- 每次状态切换都写入事件
- 甘特图只消费事件，不直接覆盖历史

## 状态切换与撤销

- 拖拽换列：视为状态切换
- 点击完成：写入 `已完成`
- 点击搁置：从 `进行中` 进入 `搁置`
- 点击恢复进行：从 `搁置` 回到 `进行中`
- 每次切换后右下角弹 `toast`
- `toast` 持续 `5 秒`
- 点击 `撤销` 时，写反向事件，不删除历史事件

## Markdown `[]` 双向绑定

定义：

- Markdown 中 `[]` / `[x]` 是正式任务节点
- 文档勾选与任务系统双向同步

规则：

- 新增 `[]`：生成子任务节点
- 勾选 `[]`：子任务进入 `已完成`
- 取消勾选 `[x]`：子任务回到 `未开始`
- 删除 checkbox 行：若任务无子节点，可软删除；否则标记废弃并保留历史

## 提醒

`v1` 支持：

- 单次提醒
- 重复提醒：每天、每周、自定义

规则：

- 提醒属于任务，不属于工作空间
- 已完成/废弃任务停止后续提醒
- 重复提醒推进下一次计划，不自动改任务状态

## 甘特图

- 位于右侧第二个 tab
- 支持按状态筛选
- 支持展示层级缩进
- 状态色与任务系统统一
- 数据来源仅为历史事件

## 桌面小窗

范围：

- 只显示 `今天进行中`
- 支持 `完成`
- 支持 `搁置`
- 支持打开详情

不支持：

- 新建任务
- 拖拽排序
- 长文档编辑

## 空态与异常态

### 今天无任务

- 中间区显示空态
- 引导快速新增

### 今天无进行中任务

- 小窗显示空态
- 提示回主窗口开始下一项

### 提醒已过期

- 仍展示原计划时间
- 标记为逾期，不自动修改任务状态
