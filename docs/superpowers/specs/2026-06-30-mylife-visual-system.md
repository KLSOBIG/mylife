# mylife 视觉系统

更新时间：`2026-06-30`

## 目标

风格参考 `Notion Calendar` 的克制感，但保持 `mylife` 自己的执行工具气质：

- 轻量
- 信息密度高
- 平面化图标
- 状态颜色清晰
- 多主题切换

## 任务状态色

- `未开始`: `#9aa4af`
- `进行中`: `#3a8d5d`
- `已完成`: `#2f7bd9`
- `废弃`: `#bb5a5a`

规则：

- 甘特条、状态圆点、任务卡侧边条、详情页状态标签统一取色
- 主题色不覆盖状态色
- hover / active 只改明暗，不改色相

## 字体与层级

- 中文优先：`PingFang SC`, `Noto Sans SC`, `sans-serif`
- 时间与数字优先：`ui-monospace`, `SFMono-Regular`, `monospace`

层级：

- 页面标题：`24/32`
- 面板标题：`16/24`
- 正文：`14/22`
- 说明文字：`12/18`
- 时间轴刻度：`12/16`

## 间距与圆角

- 页面外边距：`16`
- 面板内边距：`16`
- 卡片间距：`8`
- 分组间距：`12`
- 主圆角：`14`
- 小圆角：`10`

## 组件规则

### 任务卡

- 左侧 `4px` 状态色条
- 标题单行优先，超长省略
- 次信息包含：提醒、子任务数、更新时间
- 选中态使用主题 accent 的低饱和背景

### 状态列

- 列头展示：状态名 + 数量 + 快速新增
- 列支持展开/折叠
- 列体保持紧凑，不留大块空白

### 右侧详情

- tab 使用平面 pill
- Markdown 编辑区背景与页面 panel 区分
- 提醒配置、状态、层级关系使用紧凑表单块

### Toast

- 固定右下角
- 宽度 `280-320`
- 持续 `5 秒`
- 主文案展示动作结果
- 次操作为 `撤销`

### 甘特图

- 背景保持浅色
- 图例清楚展示状态色
- 行高紧凑，支持多层级缩进
- 时间轴弱化，状态条强化

## 图标规则

- 全部使用平面风格
- 不使用拟物阴影
- 状态图标简化为圆点、勾、斜杠、时钟
- 主题切换图标使用调色板或色块组合
